# Top Bar Menu

The Top Bar Menu component is a mobile-friendly navigation panel that slides in from the right side of the screen. It provides a structured way to organize navigation items in lists and collapsible groups.

## Component Structure

The TopBarMenu uses a compound component pattern, exposing these subcomponents:

- `TopBarMenu.Header` - Header section with a close button
- `TopBarMenu.Body` - Container for the menu content
- `TopBarMenu.List` - Groups of menu items, separated by borders
- `TopBarMenu.Group` - Collapsible groups of menu items with a label
- `TopBarMenu.Item` - Individual navigation items (links or buttons)

## Usage Example

```tsx
import { TopBarMenu } from '@reapit/elements'
import { useState } from 'react'

const MyNavigation = () => {
  const [isOpen, setIsOpen] = useState(false)

  const handleClose = () => setIsOpen(false)

  return (
    <>
      <button onClick={() => setIsOpen(true)}>Open Menu</button>

      <TopBarMenu isOpen={isOpen} onClose={handleClose}>
        <TopBarMenu.Header />
        <TopBarMenu.Body>
          <TopBarMenu.List>
            <TopBarMenu.Item href="/dashboard" isActive>
              Dashboard
            </TopBarMenu.Item>

            <TopBarMenu.Group label="Properties" hasBadge>
              <TopBarMenu.Item href="/properties">All Properties</TopBarMenu.Item>
              <TopBarMenu.Item href="/property-search">Property Search</TopBarMenu.Item>
              <TopBarMenu.Item href="/saved-properties" hasBadge>
                Saved Properties
              </TopBarMenu.Item>
            </TopBarMenu.Group>

            <TopBarMenu.Item href="/calendar">Calendar</TopBarMenu.Item>
          </TopBarMenu.List>

          <TopBarMenu.List>
            <TopBarMenu.Item href="/profile">My Profile</TopBarMenu.Item>
            <TopBarMenu.Item onClick={handleLogout}>Sign Out</TopBarMenu.Item>
          </TopBarMenu.List>
        </TopBarMenu.Body>
      </TopBarMenu>
    </>
  )
}
```
