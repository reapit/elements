# Mobile Nav Item

**A Versatile Navigation Component**

The `MobileNavItem` is a flexible component designed to render navigation items within a `MobileNav` context. It offers two primary variants:

**1. Simple Variant**

- Renders a basic navigation item using an `HTMLAnchorElement`

**2. Expandable Variant**

- Creates an expandable navigation item with a parent element (`HTMLButtonElement`) and a sub-menu (`HTMLUListElement`)
- The sub-menu contains `MobileNavSubItem` components, each rendered using an `HTMLAnchorElement`
