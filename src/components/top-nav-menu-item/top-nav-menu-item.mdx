# Top Nav Menu Item

**A Versatile Navigation Component**

The `TopNavMenuItem` is a flexible component designed to render navigation items within a `TopNavMenu` context. It offers two primary variants:

**1. Simple Variant**

- Renders a basic navigation item using an `HTMLAnchorElement` or `HTMLButtonElement`

**2. Expandable Variant**

- Creates an expandable navigation item with a parent element (`HTMLButtonElement`) and a sub-menu (`HTMLUListElement`)
- The sub-menu children is a simple variant of `TopNavMenuItem` components
