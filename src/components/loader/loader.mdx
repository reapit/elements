# Loader

A simple loader component to give a visual indicator of progress. It ships with an optional label prop as per below and can be shown inline or centered.

## Full page

The full page loader overlays any content and centers on the page - suitable for use when loading entire pages of content.

# Other attributes

`Loader` takes no custom props but all other standard HTML attributes for `<div>` are supported and are passed through to the React component, e.g. `className`
