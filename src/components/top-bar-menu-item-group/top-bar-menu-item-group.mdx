# Top Bar Menu Item Group

**An Expandable Navigation Component**

The `TopBarMenuItemGroup` is a wrapper component for `TopBarMenuItem` components that provides the ability to group multiple navigation items together in an expandable/collapsible section. It's designed for use in navigation menus, sidebars, and dropdown menus.

## Usage

```tsx
// Basic usage
<TopBarMenuItemGroup label="Reports">
  <TopBarMenuItem href="#create">Create</TopBarMenuItem>
  <TopBarMenuItem onClick={handleClick}>Generate</TopBarMenuItem>
  <TopBarMenuItem href="#archive">Archive</TopBarMenuItem>
</TopBarMenuItemGroup>

// With badge notification
<TopBarMenuItemGroup label="Notifications" hasBadge>
  <TopBarMenuItem href="#inbox">Inbox</TopBarMenuItem>
  <TopBarMenuItem href="#sent">Sent</TopBarMenuItem>
</TopBarMenuItemGroup>

// Initially expanded
<TopBarMenuItemGroup label="Settings" isActive>
  <TopBarMenuItem href="#profile">Profile</TopBarMenuItem>
  <TopBarMenuItem href="#preferences">Preferences</TopBarMenuItem>
</TopBarMenuItemGroup>
```

## Accessibility

- Menu groups are properly structured with appropriate ARIA roles
- The group button has `aria-expanded` to indicate its state
- The group button has `aria-controls` to associate it with the panel it controls
- Focus management follows established patterns for dropdown menus
