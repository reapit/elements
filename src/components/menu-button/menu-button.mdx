# MenuButton

## Styles Only Usage

The MenuButton is toggles the visibility of a Menu containing a list of buttons and/or links.

The 'top' property allows the Y position of the Menu below the button to be changed from 36px so that it can match styling of a parent component

The menu can be aligned to the left or right of the button
