Styled split button component with multiple flavours depending on use case.

A split button with no additional props / classes applied

<RenderHtmlMarkup of={SplitButtonStories.Default} />

A split menu button can be wrapped with `Menu` component.

A split button can have only two variants `primary` and `secondary`, `primary` being default.

<RenderHtmlMarkup of={SplitButtonStories.SplitButtonVariants} />

Both action and menu button can have a busy state.
