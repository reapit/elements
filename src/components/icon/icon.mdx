# Icon

The basic icon component relies on adding an `icon` prop that maps to the name of a SVG file. See the tables below for the icons available.

For non-React users, we export all the icons from `@reapit/elements/assets` where they are grouped in folders `animated-icons`, `infographic-icons`, `menu-icons` and `system-icons` eg `import addSystem from "@reapit/elements/assets/system-icons/addSystem.svg"`. You can then wrap your SVG in the appropriate classes you see in the rendered markup with each story in these docs.

<Controls />

## Basic Usage

For a basic icon, just select an icon name from one of the tables below. They are rendered in suggested sizes appropriate for their use - for details on how this works see the "Icon Sizes" story below.

> In v4 we recomment using icons with a `fontSize` prop of 24px or 1.5rem - previously the default was 16px. In version v5 24px will become the new default however in v4, we do not wish to break any existing UI using the legacy icons set so we have added a convenience `elIconNew` class to the new icon set by default. This will be removed in v5 when the default size will be 24px.
