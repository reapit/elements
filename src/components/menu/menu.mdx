# Menu

The Menu component is a panel used to display a list of buttons and/or anchors which can be ordered in Menu Groups

Each Menu Group can have a title shown above the list

More features will be added to Menu in the future

<Controls />

## Styles Only Usage

Default style of the `Menu`, can set `data-alignment` to "right" (see html of `More Complex Usage Example`) to move the x-position to the right of the button

In the React Usage, you can compose your component using `Menu` component as example here
