The following example demonstrates how to truncate text using CSS.
<Source code={`
// HTML
<div class="container">
  <span class="truncate-text">Text is truncated here</span>
</div>

// CSS styles
.container {
  width: '136px';
  display: 'flex'; // Can use grid
}
.truncate-text {
  overflow: 'hidden';
  text-overflow: 'ellipsis';
  white-space: 'nowrap'
}
`} language="tsx" />
