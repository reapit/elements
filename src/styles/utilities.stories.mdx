# Utilities

The following stories document a series of helper classes we export, primarily aimed at spacing, separating and sizing basic html tags. These helpers include margin, padding, borders, border radius and box shadow.

The helper classes follow predictable patterns so that when you become familiar with the API, you should be able to apply them without reference to the doucmentation.

### Margin & Padding

For both padding and margin, the helper classes are generated based on the following pattern:

`el` prefix +
`M` for margin or `P` for padding +
`r`, `l`, `t`, `b` for right, left top or bottom, or, `x` or `y` for left & right and top & bottom respectively. If omitted, will apply to all sides of your elemenent +
`N`, a numeric value from the table below. For margin only, as well as a numeric value `Auto` is also accepted.

### Border, Border Radius and Box Shadow

Unlike the previous properties, the border, border radius and box shadow helpers are standardised in colour and sixe around the style guide. In the case of border and border radius, you can however control the positioning of the effect or apply on all sides.

Borders come in two colours, `Blue` and `Grey`. You can also specify `R`, `L`, `T`, `B` for right, left top or bottom, or, `X` or `Y` for an axis. Not passing this value will give you a border in all sides.

Border radius can be applied to all cornders with no modifier or `BR`, `BL`, `TR`, `TL`, `B`, `T` for bottom-right, bottom-left, top-right, top-left, bottom or top respectively.

Box shadow has no modifiers and is a standardised helper.
