# Icons

Icons are simple, visual elements that help with cognitive recognition.

To use and icon as an Elements consumer, simply import the specific icon you need from it's dedicated package
entry point. For example:

```tsx
import { ChevronRightIcon } from '@reapit/elements/icons/chevron-right'
```

If consuming an icon within another Elements component, you can target the exact icon you need:

```tsx
import { ChevronRightIcon } from '#src/icons/chevron-right'
```
