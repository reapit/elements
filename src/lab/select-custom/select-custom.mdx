# SelectCustom

The **SelectCustom** component is a styled alternate for the native `<select>` element. It supports **single- and multi-select modes**, keyboard navigation, option groups, clearable selections, helper text, and chip-based display of selected values.

## SelectCustom MultiSelect

The **multi-select** mode displays each selected value as a chip, allowing users to clear selections individually or reset them all with the clear button.
